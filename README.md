## 📋 Sobre o Projeto

Landing page profissional desenvolvida para **Ademario Oliveira**, Engenheiro Civil e Mestre de Obras (CREA: 052136126-5), com foco em:

- Apresentação de serviços de construção civil e reformas
- Captação de clientes para financiamento imobiliário via **Caixa Econômica Federal**
- Solução especializada para pessoas **negativadas** que desejam conquistar a casa própria
- Formulário de orçamento integrado com canais de atendimento direto

---

## ✨ Funcionalidades

- **Hero Section** com CTA de análise gratuita
- **Processo passo a passo** — do diagnóstico à entrega das chaves
- **Cards de serviços** — Crédito para Negativados, Reformas, Gestão de Obras e Consultoria
- **Seção institucional** com perfil do profissional e especialidades
- **Depoimentos de clientes**
- **Formulário de orçamento** com seleção de tipo de serviço
- Contatos diretos via **telefone**, **WhatsApp** e **e-mail**
- Layout **100% responsivo**
